How to run ?
------
- git clone https://github.com/MrYoung249/LINE-KickBot1
- cd LINE-KickBot1 && npm install
- npm start
-----
How to rerun ?
------
- cd LINE-KickBot1
- npm start
-----
How to put your mid ?
------
- cd LINE-KickBot1 && cd src
- nano main.js
- put your mid in 'myBot'
---
**NOTE** 
- Don't forget to put your mid in main.js 'myBot'
