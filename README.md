How to run ?
------
- git clone https://github.com/MrYoung249/js1
- cd js1 && npm install
- npm start
-----
How to rerun ?
------
- cd js1
- npm start
-----
How to put your mid ?
------
- cd js1 && cd src
- nano main.js
- put your mid in 'myBot'
----
**NOTE** 
- Don't forget to put your mid in main.js 'myBot'
